# AP Police Technical Services — Staff Monitoring System

A PWA for IGP Office, Mangalagiri, Andhra Pradesh.

## Features

- **Dashboard** — Live stats, top performers scroll, heart module overview
- **Attendance** — Daily punch-in/out with WiFi validation
- **Command Register** ❤ — Work assignment tracking hub
- **Correspondence Tracker** ❤ — Drafts, Letters, Note Files, Radio Messages, eOffice
- **IGP Command Works** ❤ — Priority works allotted directly by IGP
- **Conference & Meetings Hub** ❤ — VC schedule, speakers, history
- **Staff Directory** — Full personnel directory with search
- **Leave Register** — Leave upload and balance tracking
- **Notices** — Active notice board with scrolling banner
- **Rankings** — Weekly top performers and monthly leaderboard
- **Reports & Analytics** — Downloadable attendance, task, and performance reports
- **Admin Panel** — Staff enrollment, points configuration, holiday calendar

## PWA

Installable on Android as a home screen app. Works offline after first load.

## Tech Stack

- Single-file HTML/CSS/JS PWA
- Google Sheets as backend via Apps Script relay
- Font Awesome 6.5 icons
- Inter + Rajdhani fonts
- Dark / Light mode
